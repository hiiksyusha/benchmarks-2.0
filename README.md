Graduation work

- MongoDB - это документо-ориентированная база данных, которая использует формат BSON для хранения и обмена данными. Она позволяет хранить данные различных типов, включая текст, даты и географические координаты, в документах, которые объединяются в коллекции. MongoDB также предоставляет возможность горизонтального масштабирования и распределенного хранения данных, что делает ее популярной для обработки больших объемов данных.
